MUVConf is open source cross-platform conferencing software. It provide high quality video and audio relying on modern WebRTC/HTML5 codecs. MUVConf sends video and audio data directly from user to user, so there is no needs in an intermediate media server. The number of users depends only on network bandwidth and computers performance. Compatible with most SIP servers, no special requirements.
