Configure Jitsi (SIP Communicator)

How to configure Jitsi.

To configure Jitsi, do the following:

1. Click Add new account... from File menu

2. Select iptel.org and click Advanced button

3. Enter SIP id and password, and click Next button

4. Uncheck Configure proxy automatically, enter server's IP address and port, click Next

5. Click Sign-in button
